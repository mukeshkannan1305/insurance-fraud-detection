PROJECT TITLE:
Explainable and Policy-Aware Insurance Fraud Detection

TOOLS:
Python, Streamlit, XGBoost, LIME, SHAP

RUN STEPS:
1. Install Python 3.9+
2. Open project folder
3. pip install -r requirements.txt
4. streamlit run app.py
5. Open browser at http://localhost:8501

FEATURES:
- Fraud Detection
- Policy-Aware Scoring
- Explainable AI
- Automated Claim Report
- Future Enhancement with SHAP
